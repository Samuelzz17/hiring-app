# **App Name**: JobFlow

## Core Features:

- CV Parsing: A CV parsing tool that automatically reads candidate CVs and extracts key information such as name, skills, and experience to reduce manual HR input.
- Automated Candidate Screening: An automated candidate screening tool that filters candidates based on skills, experience, and education, and provides an automated scoring and ranking system.
- Candidate Pipeline Management: Visualize and manage candidate progression through recruitment stages (Applied, Shortlisted, Interview, Hired) with an intuitive drag-and-drop interface.
- Automated Interview Scheduling: Allow candidates to automatically select available interview slots, with seamless integration with Google Calendar.
- Automated Communication: Automate sending of email communications, including interview invitations, rejection letters, and job offers.

## Style Guidelines:

- Primary color: Black (#000000) for strong, professional elements and text.
- Background color: White (#FFFFFF) to provide a clean and minimalist canvas.
- Accent color: Shades of grey (#333333, #666666, #CCCCCC) for subtle hierarchy and visual separation.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clear, and highly readable qualities, suitable for both short headings and longer descriptive text in a professional context.
- Use clean, contemporary, and professional line-based icons that intuitively represent job application stages, documents, and scheduling tasks.
- Implement a modern, clean layout with ample white space, using a card-based system for applications and a clear dashboard for an at-a-glance overview.
- Subtle and smooth transitions for state changes, form submissions, and data loading, enhancing user experience without being distracting.
- Elements and containers should feature soft, rounded corners to create a friendly and approachable feel while maintaining professionalism.
